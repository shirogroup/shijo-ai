# Shijo.ai - AI-Powered SEO SaaS Platform

Modern SEO platform with AI search optimization, keyword research, and content intelligence.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Vercel (Deployment)
- Neon (PostgreSQL)
- Claude AI (Core Intelligence)

## Phases

- Phase 1: Keyword Research (MVP)
- Phase 2: Content & On-Page SEO
- Phase 3: SERP & Competitive Insights
- Phase 4: AI Search Visibility
- Phase 5: Strategy & Workflow
- Phase 6: Rank Tracking
